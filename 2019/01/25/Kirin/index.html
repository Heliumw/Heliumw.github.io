1.基础

==以下某些内容复制自百度百科,且由于我也并不是很精通,所以尽量减少我本人的理解及观点==

  • Rec/Recovery

    ​ Recovery 是 Android 手机备份功能，指的是一种可以对安卓机内部的数据或系统进行修改的模式。

    ​ 刷入第三方的 Recovery，将获得更多的功能，并且可以刷入第三方Rom，官方自带则不行。

  • eRec/eRecovery

    eRecovery,就是 Emotion Recovery 的缩写,是华为特有的恢复模式,可以直接连 WIFI 救砖,非常方便

  • Bootloader/Fastboot

    BootLoader 深一些的刷机模式,当然我们一般也在这个模式下救砖

    Fastboot，英语翻译意思是快速启动。在安卓手机中Fastboot是一种比Recovery更底层的刷机模式（俗称引导模式）。就是使用USB数据线连接手机的一种刷机模式。相对于某些系统卡刷来说，线刷更可靠，安全。

    没看懂?没关系! 你可以理解为 Bootloader≈Fastboot

  • Hisuite

    • 就是华为手机助手
    • 设备上的数据可备份至电脑，支持联系人、短信、应用程序、音乐、通话记录、Email、日程的备份,可将已备份数据恢复至设备
    • 系统修复——救砖,且比 eRec 更底层、更彻底,同时还会锁上 Bootloader

  • 解 BL 锁

    BL 锁就是 Bootloader 锁

    因为 Bootloader 出场时被上了锁所以我们不能直接刷机,必须先解开 Bootloader 锁才行

    解锁码和方法淘宝自己搜,卖家有教程

3.正式开始

​ 前提:解开 BL 锁

  1. 基础:你需要一个 ADB 包

    下载完成后解压即可,点我下载

  2. 刷入 Recovery

    Recovery 的话,个人推荐奇兔的 REC,点我下载

    进入 ADB 解压到的文件夹(和图上差不多的文件夹)后在地址栏输入 cmd (地址栏位置请看图)并回车

    在打开的窗口中输入(此时手机处于 Fastboot 模式且将下载的7to.img放到此文件夹中)

    第二次回车同时拔掉USB线,按住音量上键直到蓝色的Honor亮起

  3. TWRP 里清Davilk,Cache,System 和 Data (只双清==也行==)

  4. 刷入Rom,Zip直接刷入,img需要刷到[ System 镜像]

  5. 按需刷入 magisk 或 gapps

  6. 格式化 Data

  7. Enjoy!
